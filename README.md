# Sane AI Blocklist

A DNS blocklist targeting consumer generative AI, LLM chatbots, and AI telemetry.

### Blocked
* Chatbots and LLMs (OpenAI, ChatGPT, Claude, Gemini, DeepSeek, Perplexity, Meta AI)
* Media and code assistants (ElevenLabs, Stable Diffusion mirrors, GitHub Copilot, Tabnine)

### Setup
1. Copy the raw URL of blocklist.txt.
2. Add the URL to your DNS filter (Pi-hole, AdGuard Home, or NextDNS).
